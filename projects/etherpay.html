Etherpay — Web SaaS

Pre-launch · Pilot ready

Mobile-first web SaaS for service-business client and billing management — track customers, charges, and payment status, with an embedded AI assistant on a Supabase backend.

Problem

Small service businesses (gyms, clinics, instructors) juggle recurring charges across many clients with whatever tool happens to be at hand — spreadsheets, paper, generic CRMs. They lose track of who paid, who is overdue, and what is due today. The workflow is read-heavy and mobile, but most CRMs are desktop-first and overbuilt for the use case.

Architecture

Next.js App Router with route groups per surface: `(app)` for the authenticated app (today, clients, charges, reports, settings) and `(auth)` for sign-up / sign-in. Supabase provides Postgres + Auth + Storage; row-level security enforces tenancy. The UI is server-component-first with client islands for interactive forms. PWA support (next-pwa) makes it installable on mobile. Migrations live under `supabase/` and are applied via `supabase db reset`. Deployed to Netlify; demo data lives behind a `refresh_demo_data(email)` Postgres function so any account can be re-seeded on demand.

Next.js (App Router), TypeScript, Supabase (Postgres + GoTrue auth + Storage + Studio), Tailwind, Radix UI primitives, react-hook-form + zod for typed forms, Playwright for E2E and Vitest + happy-dom for unit tests. PWA via @ducanh2912/next-pwa. Deployed on Netlify.

Engineering Highlights

  • Strict TypeScript end-to-end with Zod-validated forms — one schema is the source of truth for client validation, server validation, and inferred types.
  • Repeatable demo flow: a `seed.sql` plus a `refresh_demo_data(email)` Postgres function lets any tenant be re-seeded with 15 clients and ~140 charges covering every UI state (paid, pending, overdue, today, future).
  • Mobile-first PWA shell with routes per surface (`/hoje`, `/clientes`, `/cobrancas`, `/relatorios`) so owners can check today's charges from the field.
  • All-on-free-tier infrastructure (Supabase + Netlify) — the entire pilot runs at zero recurring cost.

Outcomes

  • Live on Netlify (`ether-brasil.netlify.app`) with a maintained demo account that walks recruiters and pilot users through every UI state.
  • Pilot-ready and self-deployable from the public repository, including a step-by-step setup guide.
