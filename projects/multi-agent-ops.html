Multi-Agent Operations Assistant

Production · Confidential

LangGraph multi-agent system that cut a recurring B2B operational workflow from ~1 month to ~40 minutes (~99%) for an enterprise SaaS platform.

Problem

Recurring B2B operational workflows that span multiple internal systems took roughly a month to complete because each step needed human escalation, context lookup, and manual data entry. Customer-facing teams spent the bulk of their week chasing these workflows instead of doing higher-leverage work, and the lead-time gap was visible to end customers.

Architecture

LangGraph multi-agent system: an intent classifier routes incoming requests into deterministic workflow subgraphs, and every state-changing operation is gated by a human-in-the-loop confirmation. A 'lateral query' interrupt mechanism lets users pause an in-flight workflow, ask a side question against the same agent, and resume from the exact same node without losing state. Sessions persist via TTL-bounded Redis checkpoints. Bearer-token pass-through auth keeps user credentials out of the agent process. Backend is FastAPI + async SQLAlchemy + Postgres; frontend is React 19 + TypeScript + Tailwind streaming over SSE; deployment is on AWS (ECS + MemoryDB).

Backend: FastAPI + async SQLAlchemy + PostgreSQL. Frontend: React 19 + TypeScript + Tailwind + Server-Sent Events.

Engineering Highlights

  • Deterministic workflow subgraphs replace ad-hoc agent reasoning for the steps that must be reliable; the LLM only owns intent classification and human-facing dialogue.
  • Lateral-query interrupt-and-resume pattern: pauses a running workflow, runs a side conversation, then resumes from the exact same graph node without losing state.
  • HITL confirmation gates on every state-changing operation — the agent never silently mutates a system of record.
  • TTL-bounded Redis checkpoints make sessions resumable across reconnects and bound memory growth.
  • Bearer-token pass-through auth: the agent forwards the caller's token without ever storing credentials.

Outcomes

  • ~99% cycle-time reduction on a recurring operational workflow (~1 month → ~40 minutes).
  • Production system serving the client's enterprise B2B platform; specifics under NDA.
